Snackbars

Snackbars provide lightweight feedback about an operation. They show a brief message at the bottom of the screen on mobile and lower left on larger devices. 

  var config = {
    message: "Hello World",
    buttonText: "UNDO",
    showButton: true,
    alignCenter: true,
    autoClose: true,
    multiLine: false
  };
  snackBar(config);
